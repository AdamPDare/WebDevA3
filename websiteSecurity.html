    Types of attack upon a system

    Malware

    Malicious Software (malware) is a generic term for viruses, worms, trojans and various other programs or scripts that will cause harm to a computer or gain unauthorised access to sensitive information. Malware is identified via its intended use.

    Malware tends to acquire access to a system via attaching themselves to files mainly .exe files or autorun scripts.

    Viruses

    Viruses are spread via the user copying the infected file and activating them when they are opened.

    Worms

    Worms are self-replicating and will spread incredibly quickly on the computer. Worms exploit the weaknesses in network security.

    Trojans

    Trojans are malware that is disguised as legitimate software; are installed by the user. They always have a hidden purpose and tend to run in the background.

    Logic bombs

    Logic Bombs is malicious code that was created to cause harm to a specific network, when an event occurs or when certain conditions have been met, the code is then executed. For example: an event could be a specific time or date.

    The amount of damage caused by a logic bomb does vary, but normally it causes data corruption, deletion, editing or even wiping hard drives.
